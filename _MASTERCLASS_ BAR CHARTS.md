The Journey from Data to Visual Intelligence
1. What is a Dataset?

A dataset is a collection of information arranged in rows and columns.

Example
Name	Sport	Study Hours
Aman	Cricket	2
Riya	Football	3
Anjali	Basketball	4

8. Rules of a Bar Graph
Rule 1

Data should be categorical.

✅ Sports

✅ Subjects

❌ Height

❌ Weight

Rule 2

Bars should have equal width.

Rule 3

Bars should have gaps between them.

Rule 4

Axes must be labelled.

Rule 5

Graph should have a title.

Rule 6

Y-axis should usually start from zero.

10. What is a Histogram?

A histogram is used to visualize continuous numerical data.

Instead of individual values, data is grouped into ranges.

Example
Height Range	Students
150-155	4
155-160	8
160-165	12

11. Why Do Histogram Bars Touch?

The ranges are continuous.

150-155
155-160
160-165

There are no gaps between ranges.
